# Title

## Environment

python: 3.6.13

gym: 0.15.7

mujoco-py: 2.0.2.8

This project uses [gym](https://gym.openai.com/), [mujoco-py](https://mujoco.org/), [spinup](https://spinningup.qiwihui.com/zh_CN/latest/#)

## Usage
```
python3 run_this.py
```
